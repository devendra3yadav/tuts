# Common Files

This directory contains common files used across multiple examples.

## Files

- `User.java` - User entity model
- `UserRepository.java` - User repository interface
- `UserService.java` - User service with business logic
- `UserController.java` - REST controller for User operations

## Purpose

These files are shared across different examples:
- Used in repository tests
- Used in unit tests
- Used in RestAssured tests
- Used in event examples

These are supporting files that demonstrate a complete CRUD application structure.
